# Mock flower shop

## Features

### Client side:

- Creating an account
- Login with google/mail
- Adding items to liked list
- Adding items to shopping list

Pages:

- Page for each flower
- Main discover page
- Search page

### Manager side:

- Add flowers
- Mark flower as on sale
- Adding tags
- Deleting flowers
- Clients seeing changes within a minute
